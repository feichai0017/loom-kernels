QuillSQL

A DataFusion, Arrow, Parquet, and MLIR SQL query compiler research engine. DataFusion owns SQL planning and execution; QuillSQL focuses on the JIT boundary.

Read Docs GitHub
SQL DataFusion parser, optimizer, and physical plans
Storage Arrow memory tables and Parquet datasets
Compiler MLIR lowering with melior verification
Scope Query compiler research, not OLTP storage

Storage Path

The project does not carry a private storage engine. Use DataFusion memory tables for scratch work and Parquet/Arrow datasets for persistent analytical data. 

CREATE TABLE t AS SELECT 1 AS id, 10 AS v;
INSERT INTO t VALUES (2, 20), (3, 30);
SELECT * FROM t WHERE v > 10;

MLIR Path

The JIT rule discovers DataFusion filter and projection candidates, lowers supported expressions into a small IR, emits MLIR arith functions, and verifies them through melior. 

export MLIR_SYS_220_PREFIX=/opt/homebrew/opt/llvm
export LLVM_SYS_220_PREFIX=/opt/homebrew/opt/llvm
cargo test
